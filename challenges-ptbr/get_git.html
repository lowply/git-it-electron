Git

Git é um software open source (o seu código fonte é público) escrito por Linus Torvalds, quem também criou o sistema operacional Linux.

Git é um programa para observar mudanças em arquivos ao decorrer do tempo, conhecido na programação como controle de versão. Se você já usou documentos de texto que salvavam as alterações durante a escrita, então você já está familiarizado com o conceito!

Instalar Git

Nós recomendamos instalar o Git no seu computador baixando o programa GitHub Desktop. Não iremos usar este programa durante os desafios do Git-it (nós estamos aprendendo a usar via terminal!) mas ele inclui o Git e é a forma mais fácil de instalar Git ele em todas as plataformas.

  • Windows: Use o Git Shell como terminal.
  • Mac: Abra o GitHub Desktop e, em Preferências, selecione a instalação das ferramentas de linha de comando. Use o aplicativo do terminal como o seu terminal.

Já tem Git ou não tem certeza? Escreva git --version no seu terminal e, se ele retornar um número de versão maior que 1.7.10, você está pronto! Para maiores informações, visite o website do Git.

Git não é como outros programas no seu computador. Você não irá ver um ícone na sua área de trabalho, mas sempre estará disponível para você e você poderá acessar ele a qualquer momento no seu terminal ou em aplicações que o utilizam.

Configurando o Git

Uma vez que o Git estiver instalado, abra o seu terminal. Você pode verificar que ele está lá por digitar:

git --version

Isso vai retornar a versão do Git que você está usando e aparecerá algo assim:

git version 1.9.1

A seguir, configure o Git de forma que ele saiba quem você é:

Defina seu nome:

git config --global user.name "Seu nome"

Defina seu email:

git config --global user.email "seu.email@example.com"

Símbolo de Dólar em Documentações de Códigos

Símbolos de dólar $ são usados com frequência em documentação de programação para representar que uma linha é um código da linha de comando (veja os trechos acima). Você não precisa realmente escrever o $, só o que vier após ele.
