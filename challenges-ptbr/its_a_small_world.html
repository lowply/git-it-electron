Adicione um colaborador ao seu projeto.

Plano de Fundo: Programação Social

Trabalhar com os outros é uma das melhores coisas sobre o GitHub porque torna mais fácil de trabalhar com todos em volta do mundo a qualquer momento.

Colaboradores são outros usuários GitHub que recebem permissão para fazer edições em um repositório de propriedade de outra pessoa. Contribuintes são outros usuários GitHub que fizeram contribuições para um projeto.

@reporobot é um robô do espaço sideral que ama a colaborar com repos.

Olá, Repo Robot!

Para adicionar colaboradores para um de seus projetos, visite a página do repositório em sua conta GitHub e clique na guia 'Settings' no menu na parte superior da página. Em seguida, selecione a opção 'Collaborators'. Digite o nome de usuário dentro da caixa de texto e clique em 'Add'.

Vá para sua página do repositório que você fez fork do Patchwork no GitHub e adicione 'reporobot' como um colaborador. A URL deve ser semelhante a este, mas com seu nome de usuário.

github.com/SEUUSUARIO/patchwork/settings/collaboration
