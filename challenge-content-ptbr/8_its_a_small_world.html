Adicione um colaborador ao seu projeto.

Background: Social Coding

Trabalhando com os outros é uma das melhores coisas sobre o GitHub, porque torna mais fácil de trabalhar com tods em volta do mundo a qualquer momento.

Colaboradores são outros usuários GitHub que recebem permissão para fazer edições em um repositório de propriedade de outra pessoa. Contribuintes são outros usuários GitHub que fizeram contribuições para um projeto.

@reporobot é um robô do espaço sideral que ama a colaborar em acordos de recompra.

Olá, Repo Robot!

Para adicionar colaboradores para um de seus projetos, visite a página do repositório em sua conta GitHub e clique em 'Settings' guia no menu na parte superior da página. Em seguida, selecione a opção 'Collaborators' . Digite o nome de usuário dentro da caixa de texto e clicar em adicionar 'Add'.

Vá para sua página do repositório forked Patchwork no GitHub e adicione 'reporobot' como um colaborador. A URL deve ser semelhante a este, mas com seu nome de usuário.

github.com/YOURUSERNAME/patchwork/settings/collaboration
