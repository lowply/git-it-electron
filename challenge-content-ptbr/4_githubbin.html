Trabalhando melhor, junto

O repositório que você criou até agora é apenas no seu computador, o qual é útil, mas torna-se dificil de compartilhar e trabalhar com outros. Não se preocupe, é para isso que é o GitHub!

GitHub é um website que permite que pessoas de qualquer lugar fazer o upload do que eles estão trabalhando com Git de modo que todos possam facilmente trabalhar juntos no mesmo projeto. Em vez de ter seus projetos só no seu computador, você também iria sincronizar o GitHub o qual atua como repositório central para você e todos que compartilharem.

Adicione nome de usuário GitHub para Git

Adicione seu nome de usuário GitHub para suas configurações Git. Nós vamos fazer isso só por causa do Git-it, isso torna mais fácil verificar os próximos desafios. Save exatamente como você criou no GitHub — capitalize where capitalized, e lembre-se que você não precisa entrar o "<" e ">".

Adicione seu nome de usuário GitHub à sua configuração:

git config --global user.username <USerNamE>

Você pode checar novamente o que você colocou na sua configuração Git digitando: git config --global user.username in terminal.

Tenha certeza que você digitou "user.username" acima e não "user.name", o qual iria substituir o nome que você definiu no primeiro desafio e deixa você sem propriedade de nome de usuário! Se você passou por isso, tudo bem, basta repetir o passo no primeiro desafio para adicionar seu nome e então completar o passo acima.

configuração do nome de usuário GitHub & Git não coincidem

Um erro comum é ter seu nome de usuário GitHub não conincidir com o caso de alguém que você definiu com git config. Por exemplo, 'JLord' não é o mesmo que 'jlord'

Para mudar seu nome de usuário definido com Git, basta fazer o mesmo comando que você fez anteriormente, mas com a capitalização correta:

git config --global user.username <USerNamE>

Quando você fizer suas atualizações, verifique novamente!
